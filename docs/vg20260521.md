## Approach

Several groups of DIDs in `Open3Edatapoints.py` follow strict naming and size patterns
(e.g. `EnergyConsumptionCentralHeatingMonthMatrix`, 124 bytes). Once the structure of one
DID in a group is known, all others with the same size and analogous name can be inferred
with high confidence.

**Two criteria must both be satisfied before applying a structure:**

1. **Identical byte size** — the codec size must exactly match the template.
2. **Consistent semantics** — the DID name must fit the template's concept (e.g. a DID
   named `...YearMatrix` must plausibly store per-month values for two years).

No byte-level verification against real hardware was performed for these entries.
They are marked as inferred (`vg20260521`) so they can be identified and validated later.

## Templates Used

### T1 — 124-byte MonthMatrix
**Reference DID:** 1342 `GasConsumptionCentralHeatingMonthMatrix`  
**Structure:**
```python
O3EComplexType(124, "...", [
    O3EList(62, "CurrentMonth", [O3EInt16(2, "01"), …, O3EInt16(2, "31")]),
    O3EList(62, "LastMonth",    [O3EInt16(2, "01"), …, O3EInt16(2, "31")])
])
```
31 days × 2 bytes = 62 bytes per month × 2 months = 124 bytes.

### T2 — 96-byte YearMatrix (energy / large values)
**Reference DID:** 1316 `EnergyConsumptionCentralHeatingYearMatrix`  
**Structure:**
```python
O3EComplexType(96, "...", [
    O3EList(48, "CurrentYear", [O3EInt32(4, "01_January", scale=10), …, O3EInt32(4, "12_December", scale=10)]),
    O3EList(48, "LastYear",    [O3EInt32(4, "01_January", scale=10), …, O3EInt32(4, "12_December", scale=10)])
])
```
12 months × 4 bytes = 48 bytes per year × 2 years = 96 bytes.

### T3 — 48-byte YearMatrix (gas / time / smaller values)
**Reference DID:** 1343 `GasConsumptionCentralHeatingYearMatrix`  
**Structure:**
```python
O3EComplexType(48, "...", [
    O3EList(24, "CurrentYear", [O3EInt16(2, "01_January", scale=10), …, O3EInt16(2, "12_December", scale=10)]),
    O3EList(24, "LastYear",    [O3EInt16(2, "01_January", scale=10), …, O3EInt16(2, "12_December", scale=10)])
```
